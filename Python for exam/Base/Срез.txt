Срез - способ получить часть последовательности с помощью специального синтаксиса в квадратных скобках:

	sequence[start:stop:step]

	- start - с какого индекса(включительно). Если не указан, берётся с начала последовательности.

	- stop - до какого индекса (не включительно). Если не указан, берётся до конца последовательности.

	- step - шаг (по умолчанию 1). Можно использовать отрицательные значения для обратного порядка.

В Python конструкция [::-1] используется для обратного среза последовательностей, таких как списки и строки. Она позволяет получить все элементы последовательности в обратном порядке.

Копия списка срезом: 
	
	Допустим есть список d = [1, 2, 3, 4]
	Запись d[:] означает срез списка от начала до конца, то есть копию списка.
