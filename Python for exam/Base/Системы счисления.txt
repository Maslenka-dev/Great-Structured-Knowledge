Основные системы:

	- 2 - двоичная
	- 8 - восьмеричная
	- 10 - десятичная
	- 16 - шестнадцатеричная

Запись чисел в Python

	0b1010 - двоичное → 10
	0o12 - восьмеричное → 10
	0xA - шестнадцатеричное → 10

Встроенные функции

	bin(10) - '0b1010'
	oct(10) - '0o12'
	hex(10) - '0xa'

	Возвращают строку с префиксом (0b, 0o, 0x)

int() - используется для преобразования объекта в целое число. Она может принимать один или два аргумента.

	int(value, base)

	- value - обязательный аргумент, который может быть числом, строкой или байтовым объектом.

	- base - Необязательный аргумент, указывающий систему счисления, в которой находится значение. По умолчанию — 10. Поддерживаются основания от 2 до 36 включительно. Это позволяет преобразовывать строки из различных систем счисления в десятичные целые числа.

bin() - используется для преобразования целого числа в его двоичное (бинарное) представление в виде строки.

	Префикс 0b указывает на то, что далее число идет в двоичной системе. Чтобы избавится от префикса, необходимо использовать срез: [2:].

oct() — используется для преобразования целого числа в его восьмеричное представление в виде строки.

	Префикс 0o указывает на то, что далее число идёт в восьмеричной системе. Чтобы избавиться от префикса, необходимо использовать срез: [2:].

hex() — используется для преобразования целого числа в его шестнадцатеричное представление в виде строки.

	Префикс 0x указывает на то, что далее число идёт в шестнадцатеричной системе. Чтобы избавиться от префикса, необходимо использовать срез: [2:].


Перевести число в двоичную систему счисления можно и с помощью f-строк:

		n = 13
		binary = f"{n:b}"
		print(binary)  # 1101

	Формат :b означает «вывести число в двоичной системе».

	binary = f"{n:08b}" - перевод в двоичную систему счисления с ведущими нулями (8 бит в данном случае).
