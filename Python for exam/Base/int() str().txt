str() - встроенная в Python функция, преобразующая объект в строковое представление. Функция принимает практически любые объекты

int() - встроенная в Python функция, преобразующая значение в целое число.

	int(value, base)

	- value - обязательный аргумент. Он может быть целым числом, вещественным числом, булевым значением, строкой.

	- base - Необязательный аргумент, указывающий систему счисления, в которой находится значение. По умолчанию - 10. Но поддерживаются системы счисления от 2 до 36 включительно.
