	    if x > y: return 0

Первая из них увеличивает число на экране на 3, вторая – уменьшает его на 2 (отрицательные числа допускаются).
Программа для Кузнечика – это последовательность команд.
Сколько различных чисел можно получить из числа 1 с помощью программы, которая содержит ровно 68 команд?

	from functools import lru_cache
	r = set()
	@lru_cache(None)
	def f(x, cnt):
	    if cnt == 68: 
	        r.add(x)
	        return 0
	    f(x + 3, cnt + 1)
	    f(x - 2, cnt + 1)
	f(1, 0)
	print(r)
	print(len(r))
