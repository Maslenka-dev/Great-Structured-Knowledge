Команда ping в терминале Windows — это утилита, используемая для проверки доступности узлов в сети и измерения времени, необходимого для передачи данных между вашим компьютером и удалённым сервером. Она работает по протоколу ICMP (Internet Control Message Protocol), отправляя специальные пакеты, называемые Echo Request, на указанный IP-адрес или доменное имя.

Основные функции команды ping

	- Проверка доступности: Команда позволяет определить, доступен ли удалённый хост. Если хост отвечает, это свидетельствует о его работоспособности.

	- Измерение времени отклика: После отправки пакетов ping, утилита фиксирует время, необходимое для получения ответа от хоста, что помогает оценить задержку соединения.

Команда имеет следующий базовый синтаксис:

	ping [адрес_назначения] [параметры]

Простая проверка:

	ping google.com - Эта команда отправляет 4 запроса по умолчанию и выводит результаты.

Непрерывная отправка запросов:

	ping -t google.com - Эта команда будет отправлять запросы бесконечно до тех пор, пока вы не прервете её с помощью Ctrl + C.

Изменение количества отправляемых пакетов:

	ping -n 10 google.com - Это отправит 10 пакетов вместо 4.

Изменение размера пакета:

	ping -l 1000 google.com - Эта команда отправляет пакеты размером 1000 байт.
