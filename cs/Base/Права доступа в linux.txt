Права доступа определяют, кто и что может делать с файлом или директорией. Они задаются для трех категорий пользователей:

	- u (user) - владелец файла.
	- g (group) - пользователи из группы файла.
	- o (others) - все остальные.

Виды прав

	1) r (read) - чтение:

		- файл -> можно просматривать содержимое.
		- директория -> можно видеть список файлов.

	2) w (write) - запись:

		- файл -> можно изменять/удалять.
		- директория -> можно создавать/удалять файлы внутри.

	3) x (execute) - выполнение:

		- файл -> можно запускать как программу/скрипт.
		- директория -> можно "заходить" в нее (cd).

Разберем -rwxr-xr--:

	"-" - тип объекта ("-" - файл, "d" - директория, "l" - ссылка и т.д.)

	rwx - владелец (user) может читать, писать, запускать.

	r-x - группа может читать и запускать.

	r-- - остальные могут читать.
