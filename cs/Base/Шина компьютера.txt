Шина данных - физическое соединение между функциональными блоками ПК, служащее для передачи данных между ними (система линий связи или проводников). Она служит основным каналом для обмена информацией между различными блоками компьютера. 

Последовательная шина -  тип компьютерной шины, при котором данные передаются по одному биту за раз по одной или нескольким парам проводов (каналам) последовательно (по очереди) друг за другом.

Параллельная шина - тип компьютерной шины, при которой множество битов информации передаются одновременно по раздельным проводам (линиям) шины. Каждому биту данных соответствует отдельный проводник, и все они синхронно передают свои биты за один такт. Для сравнения, последовательные шины передают данные бит за битом по одной или двум линиям, что уменьшает количество проводов и повышает дальность передачи при сохранении высокой скорости благодаря повышению тактовой частоты и более эффективному кодированию.

Примеры компьютерных шин: PCI Express, USB, SATA.
