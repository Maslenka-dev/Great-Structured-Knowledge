CLI - Интерфейс командной строки (Command line interface) - способ взаимодействия между человеком и компьютером путём отправки компьютеру команд, представляющих собой последовательность символов. Команды интерпретируются с помощью специального интерпретатора, называемого оболочкой операционной системы (shell).

GUI - Графический интерфейс пользователя (Graphical User Interface) - способ взаимодействия пользователя с компьютером с использованием графических элементов, таких как окна, кнопки и меню. 

Shell (оболочка) - программа, с которой взаимодействуют пользователи на основе текстовых комманд.

GUI (графический интерфейс пользователи) - программа, с которой взаимодействуют пользователи на основе взаимодействия с графическим интерфейсом.
