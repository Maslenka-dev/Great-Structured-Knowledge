FDDI (Fiber Distributed Data Interface - Распределенный интерфейс передачи данных по оптоволокну) — это стандарт сетевой технологии, разработанный для высокоскоростной передачи данных по волоконно-оптическим кабелям. Он обеспечивает скорость передачи до 100 Мбит/с и использует двойное кольцо для повышения отказоустойчивости: одно кольцо является основным, а другое — резервным. Если возникает проблема на одном кольце, данные автоматически перенаправляются через второе.

FDDI поддерживает два типа трафика: синхронный, для приложений, чувствительных к задержкам (например, передача видео), и асинхронный, для остальных данных. Эта технология была популярна в 1990-х годах, но с развитием Ethernet, который также предлагает высокие скорости, её использование стало менее распространённым.

Основные характеристики FDDI включают:

	- Скорость передачи: 100 Мбит/с.

	- Топология: Двойное кольцо.

	- Метод доступа: Маркерный метод управления.

FDDI также определяет протоколы физического уровня и канального уровня, что обеспечивает совместимость между различными производителями оборудования.
