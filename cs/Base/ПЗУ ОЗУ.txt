Постоянное запоминающее устройство (ПЗУ - ROM) - это тип энергонезависимой памяти, используемый в компьютерах и другой цифровой технике для хранения данных, которые не должны изменяться в процессе работы устройства.

	Основные характеристики:

		- данные сохраняются даже при отключении питания.

		- пользователь не может изменить или удалить информацию в обычных условиях эксплуатации.

		- ПЗУ используется для хранения критически важных программ и данных, необходимых для запуска и базовой работы устройства.

	Для чего нужно ПЗУ:

		- Хранение прошивок: базовые программы, управляющие работой устройства (например, BIOS или UEFI в компьютере).

		- Загрузка операционной системы: начальный этап загрузки ПК всегда начинается с обращения к ПЗУ.

Оперативно запоминающее устройство (ОЗУ - RAM) - вид энергозависимой памяти, в которой во время работы компьютера хранятся выполняемые программы и данные, необходимые процессору для текущих вычислений.

	Основные характеристики:

		- ОЗУ хранит только ту информацию, которая нужна в данный момент для работы операционной системы и запущенных программ. При выключении питания все данные стираются.

		- ОЗУ обеспечивает быстрый доступ к данным, значительно быстрее, чем жёсткие диски или SSD, что позволяет процессору эффективно обрабатывать информацию.

		- чем больше объём ОЗУ, тем больше данных и программ может одновременно обрабатывать компьютер без подвисаний.

	Работа:

		- ОЗУ служит посредником между процессором и постоянной памятью (жёстким диском или SSD), ускоряя обработку данных.

		- при запуске программ операционная система выделяет им участок оперативной памяти для хранения временных данных и кода.

		- после закрытия программы её данные освобождаются из ОЗУ для использования другими процессами.
