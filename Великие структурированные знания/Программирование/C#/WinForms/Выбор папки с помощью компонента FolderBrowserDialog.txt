В примере ниже компонент FolderBrowserDialog используется для выбора папки, аналогично ситуации, когда вы создаете проект в Visual Studio и вам предлагается выбрать папку для его сохранения. В этом примере имя папки затем отображается в элементе управления TextBox на форме. Рекомендуется поместить расположение в редактируемой области, такой как элемент управления TextBox, чтобы пользователь мог изменить свой выбор в случае ошибки или других проблем. В этом примере предполагается наличие формы с компонентом FolderBrowserDialog и элементом управления TextBox.

	public void ChooseFolder()
	{
	    if (folderBrowserDialog1.ShowDialog() == DialogResult.OK)
	    {
	        textBox1.Text = folderBrowserDialog1.SelectedPath;
	    }
	}
